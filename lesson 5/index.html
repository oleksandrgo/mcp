Server і FastMCP

MCP Server (низький рівень)

Клас Server з Python SDK — повний контроль над протоколом MCP: handlers, async, transport вручну.

  • @app.list_tools() і @app.call_tool()
  • Явний stdio_server() для transport
  • Більше коду — більше гнучкості

Server і FastMCP

FastMCP (високий рівень)

FastMCP — обгортка над Server у пакеті mcp. Python-функції стають tools через декоратори.

  • @mcp.tool(), @mcp.resource(), @mcp.prompt()
  • Схема tool — з type hints і docstring
  • mcp.run(transport="stdio") — transport «з коробки»

Server і FastMCP

Приклад: MCP Server

from mcp.server import Server
from mcp.server.stdio import stdio_server
import mcp.types as types

app = Server("my-server")

@app.list_tools()
async def list_tools() -> list[types.Tool]:
    return [
        types.Tool(
            name="add",
            description="Додати два числа",
            inputSchema={
                "type": "object",
                "properties": {
                    "a": {"type": "integer"},
                    "b": {"type": "integer"},
                },
                "required": ["a", "b"],
            },
        )
    ]

@app.call_tool()
async def call_tool(name: str, arguments: dict):
    if name == "add":
        return [types.TextContent(type="text", text=str(arguments["a"] + arguments["b"]))]
    raise ValueError(f"Unknown tool: {name}")

async def main():
    async with stdio_server() as (read, write):
        await app.run(read, write, app.create_initialization_options())

if __name__ == "__main__":
    import asyncio
    asyncio.run(main())

Server і FastMCP

Приклад: FastMCP

from mcp.server.fastmcp import FastMCP

mcp = FastMCP()

@mcp.tool()
def add(a: int, b: int) -> int:
    """Додати два числа."""
    return a + b

if __name__ == "__main__":
    mcp.run(transport="stdio")

Той самий tool add — значно менше коду. Далі працюємо саме з FastMCP.
